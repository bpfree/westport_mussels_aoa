Description:
Abstract:
This dataset provides information on the Northeastern Regional Association of Coastal Ocean Observing Systems (NERACOOS) buoys and monitoring stations. Monitoring stations in this dataset include: NERACOOS Buoys, Bowdoin Buoys, CDIP Buoys, NOAA CMAN stations, DeepC Wind, Gulf of Maine Array, Long Island Sound Array, NOAA buoys, RI-SAMP, and URI Buoys. NERACOOS is the Northeastern Regional Association of Coastal Ocean Observing Systems and is a component of the national Integrated Ocean Observing Sysem (IOOS).
Purpose:
This dataset shows locations of NERACOOS monitoring stations in the northwest Atlantic off the coast of New England and Long Island Sound

Process_Step:
Process_Description:
The buoy dataset was originally derived from a Microsoft Excel spreadsheet that included latitude, longitude, buoy, location, NOAA ID, CDIP ID, program ID, program name, status, and data access location The data were imported into ArcGIS Desktop and spatially represented using the Display XY data function, with latitude input as the Y and longitude as the X coordinate for a GCS NAD 83 geographic system.
